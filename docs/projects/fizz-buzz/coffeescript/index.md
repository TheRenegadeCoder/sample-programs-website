## Current Solution

{% raw %}

```coffeescript
for i in [1..100]   
    if i % 15 is 0  
      console.log "FizzBuzz"    
    else if i % 3 is 0  
      console.log "Fizz"    
    else if i % 5 is 0  
      console.log "Buzz"    
    else    
      console.log i
```
