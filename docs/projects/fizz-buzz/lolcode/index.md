## Current Solution

{% raw %}

```lolcode
HAI 1.2

IM IN YR loop UPPIN YR var TIL BOTH SAEM var AN 101

    DIFFRINT 0 AN var
    O RLY?
      YA RLY
        I HAS A by3 ITZ BOTH SAEM 0 AN MOD OF var AN 3
        I HAS A by5 ITZ BOTH SAEM 0 AN MOD OF var AN 5
        
        BOTH OF by3 AN by5 
        O RLY?
          YA RLY
            VISIBLE "FizzBuzz"
          NO WAI
            by3
            O RLY?
              YA RLY
                VISIBLE "Fizz"
              NO WAI
                by5
                O RLY?
                  YA RLY
                    VISIBLE "Buzz"
                  NO WAI
                    VISIBLE var
                OIC
            OIC
        OIC
    OIC
    
IM OUTTA YR loop
KTHXBYE
```
