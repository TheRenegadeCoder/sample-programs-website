## Current Solution

{% raw %}

```picolisp
(loop 
  (prinl "Welcome to the string reverser. For reversing a sentence and not just a single word, quoting with \" is required.")
  (prinl "Enter a string: ")
  (let Str (read) 
    (prinl 
      (pack
        (reverse 
          (chop D)))))
```
