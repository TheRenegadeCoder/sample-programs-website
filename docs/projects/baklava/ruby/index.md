## Current Solution

{% raw %}

```ruby
10.times {|n|
    print " " * (10 - n)
    puts "*" * (2 * n + 1)
}

for i in 10.downto(0) do
  puts ((" " * (10 - i)) + ("*" * (i * 2 + 1)))
end
```
