To run this program, download and install the latest GNU Make using these
instructions:

* [Windows][11]
* For [Linux][12], see "How to Download" section
* [Mac][13]

Download a copy of [Fizz Buzz in GNU Make][14], and run this command:

```
make -sf fizz-buzz.mk
```
