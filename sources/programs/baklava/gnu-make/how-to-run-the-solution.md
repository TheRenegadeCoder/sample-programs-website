To run this program, download and install the latest GNU Make using these
instructions:

* [Windows][12]
* For [Linux][13], see "How to Download" section
* [Mac][14]

Download a copy of [Baklava in GNU Make][15], and run this command:

```
make -sf baklava.mk
```
