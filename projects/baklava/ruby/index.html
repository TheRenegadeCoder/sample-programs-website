Baklava
in Ruby

Welcome to the Baklava in Ruby page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

HEIGHT = 10

(0..HEIGHT).each do |i|
  puts "#{" " * (HEIGHT - i)}#{"*" * (2 * i + 1)}"
end

(HEIGHT - 1).downto(0) do |i|
  puts "#{" " * (HEIGHT - i)}#{"*" * (2 * i + 1)}"
end
