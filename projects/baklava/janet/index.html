in Janet

Welcome to the Baklava in Janet page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

(for n -10 11
    (let [num_spaces (math/abs n)
        num_stars (- 21 (* 2 num_spaces))]
        (print (string/repeat " " num_spaces) (string/repeat "*" num_stars))
    )
)
