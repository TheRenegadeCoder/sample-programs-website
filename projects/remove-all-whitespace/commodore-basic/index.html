Remove All Whitespace in Commodore BASIC

Published on 17 September 2023 (Updated: 17 September 2023)
Remove All Whitespace
in Commodore BASIC

Welcome to the Remove All Whitespace in Commodore BASIC page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

5 REM Input string
10 GOSUB 1000
20 R$ = ""
30 IF S$ = "" OR S$ = CHR$(13) THEN GOTO 120
40 FOR K = 1 TO LEN(S$)
50     A$ = MID$(S$, K, 1)
60     IF A$ = CHR$(9) OR A$ = CHR$(10) OR A$ = CHR$(13) THEN GOTO 90
70     IF A$ = " " THEN GOTO 90: REM Can't put on above line, too long
80     R$ = R$ + A$
90 NEXT K
100 PRINT R$
110 END
120 PRINT "Usage: please provide a string"
130 END
1000 REM Read input value one character at a time since Commodore BASIC
1001 REM has trouble reading line from stdin properly
1002 REM S$ = string
1003 REM Initialize
1010 S$ = ""
1015 REM Append characters until end of input
1020 GET A$
1030 C = ASC(A$)
1040 IF C = 255 THEN RETURN: REM end of input
1050 S$ = S$ + A$
1060 GOTO 1020
