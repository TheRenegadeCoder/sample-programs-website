Welcome to the Remove All Whitespace in R page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

args <- commandArgs(trailingOnly = TRUE)

if (length(args) != 1 || !nzchar(args[[1]])) {
  cat("Usage: please provide a string\n")
  quit(status = 1)
}

cat(gsub("\\s+", "", args[[1]]), "\n")
