Welcome to the Fizz Buzz in Scala page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

object FizzBuzz:
  def main(args: Array[String]): Unit =
    (1 to 100).map { i =>
      (i % 3, i % 5) match
        case (0, 0) => "FizzBuzz"
        case (0, _) => "Fizz"
        case (_, 0) => "Buzz"
        case _ => i.toString
    } foreach println
