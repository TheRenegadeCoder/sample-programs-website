Welcome to the Fizz Buzz in Pony page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

use "collections"

actor Main
  new create(env: Env) =>
    for i in Range[I32](1, 101) do
      env.out.print(fizzbuzz(i))
    end

  fun fizzbuzz(n: I32): String =>
    if (n % 15) == 0 then
      "FizzBuzz"
    elseif (n % 5) == 0 then
      "Buzz"
    elseif (n % 3) == 0 then
      "Fizz"
    else
      n.string()
    end
