Josephus Problem in C#

Published on 06 November 2024 (Updated: 13 May 2026)
Josephus
Problem in C#

Welcome to the Josephus Problem in C# page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

if (
    args is not [var nText, var kText]
    || !int.TryParse(nText, out int n)
    || !int.TryParse(kText, out int k)
    || n <= 0
    || k <= 0
)
{
    Console.Error.WriteLine(
        "Usage: please input the total number of people and number of people to skip."
    );
    return;
}

int survivor = 0;
for (int i = 2; i <= n; i++)
    survivor = (survivor + k) % i;

Console.WriteLine(survivor + 1);
