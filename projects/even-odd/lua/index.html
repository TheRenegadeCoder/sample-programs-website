Welcome to the Even Odd in Lua page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

main = function(input)
    usage = "Usage: please input a number"
    
    if tonumber(input) ~= nil then
        if input % 2 == 0 then
            print("Even")
        else
            print("Odd")
        end
    else
        print(usage)
    end
end
