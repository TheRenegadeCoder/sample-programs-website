Reverse String in Vimscript

Published on 27 August 2018 (Updated: 21 December 2023)
Reverse String
in Vimscript

Welcome to the Reverse String in Vimscript page! Here, you'll find the source code for this program as well as a description of how the program works.

Current Solution

func! Reverse(str)
    return join(reverse(split(a:str, '\zs')), '')
endfunc

func! Main(...)
    echo Reverse(a:0 > 0 ? a:1 : '')
endfunc
